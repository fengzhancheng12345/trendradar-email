# TrendRadar Pro

智能股票分析订阅系统 - AI驱动的舆情监控与股票分析邮件推送服务

## 项目架构

```
网站 (index.html)  ←→  API服务 (8080)  ←→  邮件服务 (3000)
```

## 文件说明

| 文件 | 说明 |
|------|------|
| `index.html` | 前端网站 - 登录/注册/用户中心/自选股 |
| `api_service.py` | Python API服务 - 用户管理/股票CRUD/会员权限 |
| `trendy-v4.cjs` | Node.js邮件服务 - 生成AI分析报告并发送 |

## 功能

- ✅ 用户注册/登录
- ✅ 自选股票管理
- ✅ 会员等级系统（免费3次/高级/超级）
- ✅ AI分析报告邮件推送
- ✅ 实时热搜监控
- ✅ 大盘指数数据

## 快速部署

```bash
# 1. 启动API服务 (8080端口)
python api_service.py

# 2. 启动邮件服务 (3000端口)
node trendy-v4.cjs
